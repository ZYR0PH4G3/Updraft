This is a game similar to flapy bird in which you have to avoid moving obstacles to set the highest score possible. To play the game all you have too do is press the space bar however you must time it very carefully if you  wish to survive through the obstacles ahead. There are moving obstacles which I made and a rocketship which rises vertically upward. It is a fun fast paced game which tests your reaction times as well as how well you cann time your moves.
In this project I was unable to make very good game art because of some issues with aseprite which hopefully will be resolved soon. I followed some youtube tutorials for the game structure and some tricks to make my works go faster. It helped me learn a lot about coding in dscript and godot as a whole and I also got to larn about some new nodes.
How to play the game: 
Well its simple just click start game and start pressing the spacebar.
